🚀 E-Blog – Django Blog Web Application

A full-stack blog web application built with Django that allows users to register, authenticate, create blog posts, manage posts, and interact through comments.

This project demonstrates Django Class-Based Views, authentication, CRUD operations, form validation, and relational database usage.

📌 Project Description

E-Blog is a simple blogging platform where users can:

  1. Register and login securely
  2. Create and publish blog posts
  3. Update and delete posts
  4. View posts from other users
  5. Like, Dislike and Comment on blog posts

The project is designed to demonstrate core backend development concepts using Django.

🧠 Key Concepts Demonstrated

  1. Django Class-Based Views
  2. Django Authentication & Authorization System
  3. Admin Post Approval
  4. CRUD Operations
  5. Form Handling
  6. Data Validation using Regex
  7. Database Relationships
  8. Like, Dislike and Comment System
  9. User Session Management
  10. User Post View
  11. Searching the post
  12. Template Rendering
  13. Pagination Implemented
